About toughtongue-skills

Agent skills and MCP server for ToughTongue AI, the voice-agent platform for high-stakes conversation practice: interviews, sales, coaching, negotiations.

Install once, then work with ToughTongue from Claude Code, Codex, Cursor, or any agent that supports the Agent Skills format:

- "Pull the last 3 lost deals from our call notes and create a practice scenario for the pricing objection." - "For the Enterprise Discovery Call scenario, pull the last 50 sessions. What are the top 5 improvement areas?" - "The onboarding agent keeps ending calls too early. Pull the low-scoring sessions, find out why, and fix the scenario."

These compose with the rest of your MCP ecosystem: Gong, Notion, calendar, slides, email. See What you can do for the full journeys.
